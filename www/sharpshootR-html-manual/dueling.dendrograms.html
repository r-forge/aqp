Dueling Dendrograms

Description

Graphically compare two related dendrograms

Usage

dueling.dendrograms(d.1, d.2, lab.1 = "D1", lab.2 = "D2", cex.nodelabels=0.75, arrow.length=0.05)

Arguments

d.1

left-hand dissimilarity matrix, re-scaled to [0,1]

d.2

right-hand dissimilarity matrix, re-scaled to [0,1]

lab.1

left-hand title

lab.2

right-hand title

cex.nodelabels

character expansion size for node labels

arrow.length

arrow head size

Details

Dissimilarity matrices are converted into dendrograms via divisive hierarchical clustering via diana(). Connector arrows are used to link nodes from the left-hand dendrogram to the right-hand dendrogram.

Author(s)

D. E. Beaudette
